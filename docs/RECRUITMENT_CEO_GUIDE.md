# Recruitment Feature - CEO Quick Start Guide

## Overview
This guide helps you quickly initiate and manage recruitment for your two positions.

## üöÄ Quick Start (5 Minutes)

### Step 1: Create Recruitment Request
1. Navigate to **Recruitment** ‚Üí **New Position**
2. Fill in position details:
   - **Position Title**: e.g., "Senior Software Engineer"
   - **Department**: e.g., "Engineering"
   - **Hiring Manager**: Select yourself (CEO)
   - **Priority**: Choose "High" or "Urgent"
   - **Headcount**: Number of positions (usually 1)
   - **Employment Type**: Full-time, Contract, etc.

3. **Job Details**:
   - Job Description: Copy from your job posting
   - Requirements: Key qualifications
   - Required Skills: List essential skills (used for CV scoring)
   - Salary Range: Min and Max (helps filter candidates)

4. Click **Create Request**

‚úÖ **Result**: You'll get a unique Requisition Number (e.g., `RRF-20260121-0001`)

---

### Step 2: Add Candidates
You have 3 options:

#### Option A: Upload CV (Recommended)
1. Go to your requisition
2. Click **Add Candidate**
3. Enter basic info (name, email, phone)
4. Click **Upload CV**
5. System automatically:
   - Extracts information
   - Scores CV against job requirements
   - Ranks candidates

#### Option B: Manual Entry
1. Click **Add Candidate**
2. Fill in candidate profile manually
3. Save

#### Option C: Bulk Import (Coming Soon)
- Import multiple candidates from CSV

---

### Step 3: Screen & Shortlist
1. View all candidates under your requisition
2. System shows:
   - **CV Score**: Overall match % (0-100)
   - **Skills Match**: How well skills align
   - **Experience Level**: Years of experience
3. Click candidate to see detailed profile
4. Move promising candidates to **Screening** stage

**Quick Tip**: Sort by CV Score to see best matches first

---

### Step 4: Schedule Interviews
1. Select candidate ‚Üí Click **Schedule Interview**
2. Choose interview type:
   - Phone Screen (15-30 min)
   - Technical Interview (60-90 min)
   - Manager Interview (45-60 min)
   - Panel Interview
3. **Provide Time Slots**:
   - Add 2-3 available time slots
   - System sends to candidate
   - Candidate selects preferred time
4. System automatically:
   - Sends confirmation email (24 hours before)
   - Updates your calendar

---

### Step 5: Track Progress
View your **Manager Dashboard** to see:
- Total candidates per position
- Shortlisted candidates
- Interviewed candidates
- Days since request opened

## üö® Troubleshooting

### "Upload Failed"
- Check file size < 10 MB
- Ensure file is PDF, DOCX, or DOC
- Try again or contact HR

### "No CV Score"
- CV may not have clear structure
- Add skills manually
- HR can review and score manually

### "Candidate Not Responding"
- Check email spam folder
- System auto-reminds after 3 days
- HR can send manual follow-up

### "Duplicate Requisition Number"
- This should not happen (fixed!)
- If it does, contact support immediately
- Use the **latest** requisition
